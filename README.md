# DDEV Desktop Releases

AppImage may require [Fuse]([url](https://github.com/appimage/appimagekit/wiki/fuse)).
```
sudo apt-get install -y fuse
```

> [!NOTE]
> You may need to run the AppImage with the `--no-sandbox` flag
